# 한글 서바이버 샘플

스마트폰 세로 화면에서 바로 실행되는 `뱀파이어 서바이벌`류 한국어 노출 게임 샘플입니다.

## 핵심 루프

- 화면을 드래그해서 캐릭터를 이동합니다.
- 캐릭터는 가장 가까운 적을 자동으로 공격합니다.
- 적은 스테이지별 테마에 맞는 부정적인 한국어 단어들입니다.
- 아이템은 `회복`, `돈`, `경험`, `보호`처럼 긍정적이거나 도움이 되는 한국어 단어들입니다.
- 드물게 `자석`, `변신`, `청소` 같은 희귀 아이템이 떨어져 전투 흐름을 크게 바꿉니다.
- 일반 적은 `일반형`, `탱커형`, `무리 속도형`으로 나뉘고, 무리 속도형은 여러 마리가 함께 빠르게 몰려옵니다.
- 적은 여기에 더해 `돌진형`, `원거리형`, `고함형`, `증식형` 같은 행동 역할을 가질 수 있습니다.
- 레벨업 선택지는 `힘`, `속도`, `건강`, `집중`, `파동`, `불꽃`, `번개`처럼 실제 한국어 단어로 표시됩니다.
- 레벨업처럼 전투가 잠시 멈추는 장면에서는 작은 베트남어 뜻을 함께 보여 어려운 단어 의미를 바로 짐작할 수 있습니다.
- 90초씩 `감정 -> 건강 -> 인생` 3개 스테이지가 이어지고, 단계가 올라갈수록 적이 강해집니다.
- 각 스테이지마다 20초 전후로 테마에 맞는 미니보스 단어가 등장합니다.
- 에픽 몬스터는 한국어 부정 단어 발사체로 원거리 공격도 해옵니다.
- 에픽 몬스터와 발사체는 `절망`, `혼돈`, `공포`, `좌절`, `압박`, `상처` 같은 이름에 맞는 형태와 움직임으로 다르게 보입니다.
- 3번째 스테이지를 버티면 최종 보스 `파멸`이 등장하고, 다양한 강공격을 넘겨야 완전 클리어할 수 있습니다.
- 클리어 시 받은 데미지, 준 데미지, 먹은 아이템이 로그로 저장되고 첫 화면에서 랭킹과 상세 기록을 볼 수 있습니다.
- 로그와 랭킹에는 몇 개 스테이지를 돌파했는지와 지나온 테마가 함께 남습니다.
- 전투 중 `각성` 게이지가 차면 잠시 더 빠르고 강해지며, HUD와 화면 연출로 발동 상태를 바로 알 수 있습니다.

## 폴더

- `index.html`: 실행 파일
- `styles.css`: 모바일 전체 화면 스타일
- `src/data/vocab-data.js`: 적, 아이템, 강화명 데이터
- `src/scenes/arena-scene.js`: 실제 게임 루프

## 샘플 목적

현재 버전은 아래를 검증하기 위한 플레이어블 샘플입니다.

- 스마트폰 터치 조작이 자연스러운지
- 학습 부담 없이 한국어 노출이 자연스러운지
- 생존형 액션 자체가 재미있게 성립하는지
- 보스, 스테이지, 무기, 보상 시스템이 뱀서 느낌을 잘 살리는지
