# 한글 서바이버 샘플

스마트폰 세로 화면에서 바로 즐기는 `뱀파이어 서바이버`류 한국어 노출 게임 샘플입니다. 퀴즈를 맞히기보다, 전투 중에 한국어 단어를 계속 보고 듣듯 익숙해지는 흐름을 목표로 합니다.

## 현재 전투 흐름

- 가상 조이스틱으로 이동하고 공격은 자동으로 발동합니다.
- 적은 부정적 의미의 한국어 단어로 나타납니다.
- 아이템과 강화는 긍정적이거나 도움이 되는 한국어 단어로 구성됩니다.
- 각 스테이지는 `90초` 동안 진행됩니다.
- `30초`, `60초`마다 강력한 강습 웨이브가 등장합니다.
- `90초`가 지나면 남은 적을 모두 정리한 뒤 스테이지 보스가 등장합니다.
- 스테이지 보스를 쓰러뜨리면 `전설 보상 5개 중 1개`를 선택합니다.
- `1단계 감정 -> 2단계 건강 -> 3단계 인생` 순서로 진행됩니다.
- 3스테이지 보스를 돌파하면 전설 보상을 하나 더 고른 뒤 최종 보스 `파멸`과 결전합니다.
- 클리어 기록은 랭킹에 남고, 과거 로그를 눌러 받은 데미지, 준 데미지, 먹은 아이템 등을 확인할 수 있습니다.

## 적과 보상

- 일반 적은 `일반형`, `탱커형`, `무리 속도형`으로 나뉩니다.
- 행동 역할로 `돌진형`, `원거리형`, `고함형`, `증식형`이 섞여 전장을 압박합니다.
- 희귀 드롭으로 `자석`, `변신`, `청소`가 등장합니다.
- 스테이지 보스 보상으로 `거인`, `폭풍`, `혜성`, `풍요`, `초신성` 중 하나를 고를 수 있습니다.

## 주요 파일

- `index.html`: 실행 진입점
- `styles.css`: 모바일 화면 레이아웃
- `src/data/vocab-data.js`: 어휘, 스테이지, 보상 데이터
- `src/scenes/arena-scene.js`: 전투 루프와 HUD, 보스/웨이브 흐름
- `src/scenes/menu-scene.js`: 시작 화면과 랭킹 UI
