문법 학습

-(으)ㄴ/는 모양이다

상황을 보고 추측할 때 사용해요.

Tip! '-(으)ㄹ 것 같다'와 비슷하지만, '-(으)ㄴ/는 모양이다'는 직접 보거나 들은 근거가 있을 때 더 자연스러워요.

형용사 (Adjective)

  • 받침 X / ㄹ -ㄴ 모양이다
    바쁘다 → 바쁜 모양이다
  • 받침 O -은 모양이다
    작다 → 작은 모양이다
  • Exception (있다/없다) -는 모양이다
    맛있다 → 맛있는 모양이다

동사 (Verb)

  • Present (현재) -는 모양이다
    가다 → 가는 모양이다
    먹다 → 먹는 모양이다

명사 (Noun)

  • Present (현재) -인 모양이다
    학생 → 학생인 모양이다

시제 (Tense)

  • Past (과거) -았/었던 모양이다
    비가 많이 왔던 모양이다
  • Future (미래) -(으)ㄹ 모양이다
    비가 올 모양이다
