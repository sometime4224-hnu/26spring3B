# hnu-3B-korean-webapp

한국어 학습용 웹페이지 모음입니다.

## 현재 구조

- `index.html`: 통합 메인 허브
- `c10` ~ `c17`: 챕터별 학습 페이지
- `c18`: 18과 준비 페이지
- `review4`, `review5`, `review6`, `finalreview`: 복습/평가
- `ex`: 확장 연습 자료
- `writeclass`: 쓰기 수업 자료
- `supplement`: 루트 산재 자료를 주제별로 정리한 폴더
  - `supplement/tools`: 수업 운영/도구
  - `supplement/games`: 게임형 활동
  - `supplement/tests`: 테스트/실습
  - `supplement/reading`: 읽기/문화 보충
- `archive/c13_backup`: 중복 보관본(`_c13`) 아카이브

## 정리 원칙

1. 루트에는 허브 성격 파일만 둡니다.
2. 기능/주제별 폴더에 페이지를 묶어 관리합니다.
3. 챕터 인덱스에서 누락된 링크는 임시 준비 페이지로 연결해 동선을 끊지 않습니다.
